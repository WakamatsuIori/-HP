# 手順書02: 故障通知用のDiscord Webhookを作る

所要時間: 約5分。

## なぜ必要？

HPの自動更新が失敗したとき（カレンダー取得失敗・ビルド失敗など）に、
Discordの自分専用チャンネルへ自動で知らせるためです。
「気づかないうちにHPの更新が止まっていた」という放置事故を防ぎます。

## 手順

### 1. 通知を受け取るチャンネルを用意する

自分だけのサーバーがなければ作ります：

1. Discordの左側の **＋（サーバーを追加）** → **オリジナルの作成** → 「自分と友達のため」
2. サーバー名は「HP管理」など好きな名前でOK
3. テキストチャンネルを1つ作る（例：`#hp-アラート`）

### 2. Webhookを作る

1. 通知用チャンネルの **⚙（チャンネルの編集）** をクリック
2. **連携サービス** → **ウェブフック** → **新しいウェブフック**
3. 作成されたWebhookをクリックして開き、名前を「HP監視」などに変更（任意）
4. **ウェブフックURLをコピー** をクリック

### 3. URLを控える

コピーしたURL（`https://discord.com/api/webhooks/～`）は後ほど
**GitHubのSecrets**（手順書03）に登録します。それまでメモ帳などに一時保管してください。

## ⚠️ 重要な注意

- **このURLは秘密情報です。** 知っている人は誰でもそのチャンネルに投稿できてしまいます
- X等に貼らない、コードに書かない、`.env` と GitHub Secrets 以外に保存しない
- 万一漏れた場合は、同じ画面からWebhookを削除して作り直せば無効化できます
