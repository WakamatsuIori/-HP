# 手順書01: 配信予定専用のGoogleカレンダーを作る

所要時間: 約5分。PCのブラウザで行ってください。

## なぜ必要？

このHPは「カレンダーに予定を入れるだけで、サイトのスケジュールが自動更新される」仕組みです。
普段使いのカレンダーとは別に、**配信予定だけを入れる専用カレンダー**を1つ作ります
（専用にするのは、プライベートの予定がHPに公開されるのを防ぐためです）。

## 手順

### 1. 専用カレンダーを作る

1. PCで [Googleカレンダー](https://calendar.google.com/) を開く
2. 左側の「他のカレンダー」の右にある **＋** をクリック → **新しいカレンダーを作成**
3. 名前に「**配信予定**」と入力（説明は空欄でOK）
4. **カレンダーを作成** をクリック

### 2. カレンダーを公開する

1. 作成後、左の一覧に「配信予定」が増えているので、その上にマウスを乗せて **︙（縦3点）→ 設定と共有** をクリック
2. 「予定のアクセス権限」の **「一般公開して誰でも利用できるようにする」にチェック**
3. 「危険です」という警告が出ますが、**このカレンダーには配信予定しか入れない**ので **OK** をクリック

### 3. .ics のURLを取得する

1. 同じ設定画面を下にスクロールし、「**カレンダーの統合**」を探す
2. 「**iCal形式の公開URL**」の欄のURLをコピー（`https://calendar.google.com/calendar/ical/～/public/basic.ics` のような形）

### 4. プロジェクトに設定する

1. プロジェクトフォルダ（`vtuber-hp`）にある `.env.example` をコピーして、同じ場所に `.env` という名前で保存
2. `.env` をメモ帳などで開き、`ICS_URL=` の後ろにコピーしたURLを貼り付けて保存

```
ICS_URL=https://calendar.google.com/calendar/ical/xxxxx/public/basic.ics
```

### 5. 動作テスト

1. 「配信予定」カレンダーに、テストの予定を1つ入れる（例：明日21:00「テスト配信」）
2. ターミナルで `npm run dev` を実行し、表示されたURL（通常 http://localhost:4321）を開く
3. トップページと「スケジュール」ページにその予定が表示されればOK！

## 運用のコツ

- 予定のタイトルがそのままHPに表示されます（例：「ゲーム配信」「歌枠」）
- 毎週決まった曜日の配信は「繰り返し」設定でOK。お休みの週はその回だけ削除すれば、HPからも消えます
- **このカレンダーは全世界に公開されます。配信予定以外を入れないでください**
