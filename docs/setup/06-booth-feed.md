# 手順書06: BOOTHのフィードURLを用意する

所要時間: 約5分。PCのブラウザで行ってください。

## なぜ必要？

「グッズ」ページに、BOOTHの商品を自動で掲載するために、BOOTHショップの **RSSフィードURL** を使います。
HPはショーケース（見せるだけ）で、**購入は各商品のBOOTHページで行います**（当サイトでは販売しません）。

## 手順

### 1. 自分のBOOTHショップURLを確認する

- 自分のBOOTHショップのトップを開く（`https://〇〇.booth.pm/` の形。`〇〇` がショップ名）

### 2. フィードURLを作る

- ショップURLの末尾に `items.rss` を付けたものがフィードURLです
  ```
  https://〇〇.booth.pm/items.rss
  ```
- このURLをブラウザのアドレス欄に入れて開き、**商品一覧のデータ（XML）が表示されるか確認**してください
  - 文字がたくさん並んだページ（`<item>` などのタグ）が出ればOK
  - エラーや「見つかりません」が出る場合は、ショップ名の綴りを確認してください

### 3. プロジェクトに設定する

- `src/config/site.ts` の `sources.boothFeedUrl` に貼り付け
  ```ts
  sources: {
    ...
    boothFeedUrl: 'https://〇〇.booth.pm/items.rss',
  }
  ```
  ※ フィードURLは公開情報なので、ここ（コミットされるファイル）に書いてOKです。

### 4. 動作テスト

1. ターミナルで `npm run dev` を実行し、表示URLを開く
2. 「グッズ」ページ（/goods）に商品が並べばOK
3. まだ商品が無いショップの場合は「準備中」と表示されます（正常です）

## 注意（重要）

- BOOTHは、ブラウザ以外からの自動アクセスを制限していることがあります。
  そのため、**自動ビルド（サーバー）からフィードを取得できない可能性**があります。
- まずは上記「2.」で **ブラウザでRSSが開けるか** を確認してください。
- もし本番の自動ビルドでグッズが取得できない場合は、取得方法の調整が必要になることがあります。
  その時はお知らせください（ヘッダー調整や代替の取得方法を検討します）。
