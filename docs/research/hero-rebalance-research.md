# ヒーロー（ファーストビュー）リバランス調査＋再提案

> 課題：ユーザー指摘「ヒーローの文字とリンクのバランスが悪い／フォロー導線が多く“売りつけすぎ”」。競合VTuber公式HPのヒーローをクロール調査し、UX一次資料と突き合わせて、PC・スマホ両方のヒーローを再設計する。

---

## 1. 現状ヒーローの実測（スマホ 390px / PCはCSSから算出）

上から：①小見出し「個人勢VTuber ・ Cafe & Bar」(11.8px) → ②屋号「和香松庵」(48px / PCは最大83px) → ③ローマ字「WAKAMATSU IORI」(13px・字間広め) → ④**2行キャッチ**「雑談とゲームを肴に、ゆるりと楽しい夜を。ようこそ、隠れ家カフェ＆バーへ。」(15.7px・34字) → ⑤ライブバッジ（配信時のみ表示・現在は非表示） → ⑥**カウントダウンカード**「次の配信まで 19時間13分 / Forza Horizon 6」(配信が遠くても常時表示) → ⑦**CTAボタン3つ**（YouTubeチャンネルへ＝金／Xをフォロー＝線／今週の予定を見る↓＝線）。スマホでは**3つとも全幅(345px)・各56〜58pxで縦積み**。

**問題点**：折りたたみ上に「48px見出し＋ローマ字＋2行コピー＋カウントダウン＋全幅ボタン3連」と要素が密集。特に**全幅ボタン3連**が“売り込み感”の主因。カウントダウン常時表示は煽りに転びやすい。

## 3. ファーストビューUXの一次資料（要点）

- **主CTAは1つに集中**。複数CTAは“選択のパラドックス／choice overload”でコンバージョンが分散。補助は最大1つ、視覚的に明確に従属させる（色を抑える・アウトライン/テキスト化）。 [Telerik](https://www.telerik.com/blogs/how-many-ctas-are-too-many) / [CXL](https://cxl.com/blog/does-offering-more-choices-actually-tank-conversions/) / [GetUplift](https://getuplift.co/the-psychology-of-a-cta-button/)
- **スキャン読み（F/Z型）**。見出しは先頭2語で要旨が伝わるよう front-load、コピーは短く。 [NN/g F-pattern](https://www.nngroup.com/articles/f-shaped-pattern-reading-web-content/)
- **信頼の階層**：①自分に関係あるか→②他より良いか→③個人情報/コミットの価値があるか の順。**下位の信頼を満たす前にフォロー/登録を前面化しない**（急かすと厚かましい印象）。 [NN/g Hierarchy of Trust](https://www.nngroup.com/articles/commitment-levels/)
- **モバイル**：主CTAは親指圏（下〜中央下）に全幅、タップ領域**最小44px**、本文**16px以上**、折りたたみ上は「見出し・価値・主CTA」の3点に絞る。 [WebAbility 44px](https://www.webability.io/glossary/target-size) / [thumb zone](https://heyflow.com/blog/mastering-the-thumb-zone/) / [above the fold](https://hyperspeed.me/blog/above-the-fold-optimization/)
- **ソーシャルプルーフは1点だけ・盛らない**。数字が弱いなら出さず世界観で信頼を作る。金は装飾/見出し/線に限定（本文に金はコントラスト不足）。 [NN/g social proof](https://www.nngroup.com/articles/social-proof-ux/) / [A11Y Pros](https://a11ypros.com/blog/accessibility-in-design-systems)
- **煽り装置は常設しない**：カウントダウンは配信近接時のみ、「今すぐ/急げ」系の文言を避ける。 [NN/g commitment](https://www.nngroup.com/articles/commitment-levels/)

## 4. ★ 再提案：ヒーロー新構成

### 方針
**「肩書き → 屋号 → ローマ字 → 短いキャッチ → 主CTA金1つ → 控えめな予定への誘導」までをヒーローに。フォロー連打・常時カウントダウン・ライブ常設は下段へ逃がす。**（夢乃ほのか／理芽／hololiveの“上品でちょうどいい”型）

### 現状①〜⑦の判定
| # | 要素 | 判定 | 内容 |
|---|---|---|---|
| ① | 小見出し（個人勢VTuber・Cafe&Bar） | **残す（小さく）** | 屋号の説明として控えめに維持 |
| ② | 屋号「和香松庵」 | **残す** | 主役・最大 |
| ③ | ローマ字 | **残す（小さく1回）** | 黒金系の定石。装飾として小 |
| ④ | 2行キャッチ | **1行に圧縮（核を前置）** | 例「ようこそ、隠れ家カフェ＆バーへ。」など20字前後（※コピーは要本決め・仮） |
| ⑤ | ライブバッジ | **配信時のみ・控えめ** | 現状も配信時のみ表示＝維持でOK |
| ⑥ | カウントダウン | **ヒーローから下げる**（SCHEDULE直上 or 配信48h以内のみ表示） | 常時表示の煽りを解消 |
| ⑦ | CTA 3つ | **主1＋誘導1に削減** | **主＝「YouTubeチャンネルへ」金ボタン1つ**／**「今週の予定を見る↓」はボタンでなく控えめなテキスト＋矢印リンクに降格**／**「Xをフォロー」はヒーローから削除**（FOLLOWセクション＋フッターに既存） |

### PC版（Z型）
左上：小見出し＋屋号＋ローマ字 → 中央：1行キャッチ → 左下：**主CTA金「YouTubeチャンネルへ」1つ** ＋ その下に小さく「今週の予定を見る ↓」テキスト誘導。立ち絵は右で視線を内側へ。

### スマホ版（縦積み・折りたたみ上は3点）
屋号 → 1行キャッチ → **主CTA金（全幅）「YouTubeチャンネルへ」**。その下に控えめテキスト「今週の予定を見る ↓」。ライブ/カウントダウンは配信近接時のみ＝通常はヒーローに出さない。本文16px以上、CTA高さ44px以上。

### 期待効果
- ボタンが**3連→1つ**になり「売りつけ感」が解消。
- フォロー導線はヒーローから消えるが、**FOLLOWセクション＋フッターSNS**で十分担保（むしろ重複が減る）。
- 折りたたみ上の密度が下がり、屋号・世界観・主CTAが際立つ。

### ユーザーに決めてほしい点
1. **主CTAはYouTubeでよいか**（配信が入口＝推奨）。それとも「予定/スケジュール」を主にするか。
2. **カウントダウン**：完全にSCHEDULE直上へ移すか／「配信48時間以内だけヒーローに出す」か。
3. **キャッチ1行案**の文言（仮。本決めはユーザー）。
